User flows
👨‍🎓 Student (public)
Browse & contact PGs
  • Lands on homepage
  • Filters by price, gender, furnishing
  • Views listing detail + photos
  • Clicks "Contact on WhatsApp" or calls
  • No login required to browse
🔐 Admin (private)
Manage all listings
  • Logs in at /admin/login
  • Adds new PG listing + uploads photos
  • Edits / deletes listings
  • Marks listings as verified
  • Manages enquiry visibility

Smart MVP decisions
✓ No student login needed (reduces friction on day 1)
✓ WhatsApp button = free, instant conversion
✓ Admin-only listing control = quality stays high
✓ Cloudinary free tier = 25GB, plenty for MVP
→ Start with 10 real listings, not fake data
→ Add "Verified by admin" badge for trust
→ Distance from Presidency gate is a killer filter
→ Monetise later: charge ₹500–2000/month to list
